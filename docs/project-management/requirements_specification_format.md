## Condiciones

**Precondición:**
Estado o condición que debe existir antes de que este requisito pueda ejecutarse.

**Postcondición:**
Estado resultante del sistema luego de que el requisito se cumple correctamente.

## Prioridad

| Nivel  | Descripción                                      |
|--------|--------------------------------------------------|
| Alta   | Esencial para el MVP, bloquea otras funciones    |
| Media  | Importante pero no bloquea el flujo principal    |
| Baja   | Deseable, puede diferirse a una versión futura   |

**Prioridad asignada:** Alta / Media / Baja
