Este archivo contiene los cuerpos (Body) y métodos probados y funcionando para
cada endpoint.

RECOMENDACIONES GENERALES:
- Los Enums se mandan como STRING (Ej: "French oak", "Red", "Legendary").
- Ojo con los campos obligatorios como 'medalCount' en las métricas.
- Para relaciones, manden el objeto con su ID (Ej: "country": { "id": 1 }).

-------------------------------------------------------------------------------
1. PAÍSES (Country)
-------------------------------------------------------------------------------
MÉTODO: POST
URL: /api/countries
BODY:
{
  "name": "South Africa",
  "code": "ZA"
}
