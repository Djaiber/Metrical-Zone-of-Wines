1. PAISES (Country) -> Deberia quedar con ID 11
-------------------------------------------------------------------------------
METODO: POST
URL: /api/countries
BODY:
{
  "name": "Germany",
  "code": "DE"
  // OJO: 'code' debe ser exacto de 2 letras y UNICO.
}
